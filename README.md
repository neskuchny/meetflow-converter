## ✨ Что умеет

- **🔄 Смена формата** — видео (MP4, MKV, WebM, MOV, AVI…) и аудио (MP3, WAV, AAC, OGG, FLAC, M4A).
- **🎧 Звук из видео** — вытащить аудиодорожку из любой записи в один клик.
- **🗜 Сжатие** — до **целевого размера** (25 / 50 / 100 / 250 / 500 МБ) или по качеству, чтобы файл влез в почту / мессенджер.
- **📦 Массовая обработка** — выбрать сразу **несколько файлов или целую папку** → результат в выбранную папку, с общим прогрессом.
- **🧩 Сшить (склейка)** — собрать несколько роликов в один; **порядок задаётся стрелками** ↑↓; разные форматы и разрешения нормализуются автоматически.
- **✂️ Разбить (нарезка)** — на N равных частей или по длительности (минуты), с точными разрезами.
- **🔊 Звук** — нормализация и **усиление громкости**, **шумоподавление** фона; видео при этом сохраняется.

Всё работает **офлайн** благодаря встроенному движку ffmpeg — ничего доустанавливать не нужно.

## 🚀 Установка и первый запуск

<details>
<summary><b>🪟 Windows</b></summary>

1. Скачайте `.exe` и запустите.
2. При первом запуске Windows SmartScreen может показать синее окно «Защита Windows».
   Нажмите **«Подробнее» → «Выполнить в любом случае»**.
   Это разовое предупреждение, потому что установщик пока без платной подписи — приложение безопасное и работает локально.
</details>

<details>
<summary><b>🍎 macOS</b></summary>

1. Откройте `.dmg` и перетащите **MeetFlow Converter** в «Программы».
2. При первом запуске: правый клик по иконке → **«Открыть»** → ещё раз **«Открыть»**.
   (Так macOS Gatekeeper пропускает приложение без подписи разработчика.)
</details>

<details>
<summary><b>🐧 Linux</b></summary>

**AppImage:** дайте файлу право на запуск и запустите —
```bash
chmod +x MeetFlow*.AppImage
./MeetFlow*.AppImage
```
**.deb (Debian/Ubuntu):**
```bash
sudo apt install ./meetflow-converter_0.2.0_amd64.deb
```
</details>

## 🔒 Приватность

Все файлы обрабатываются **только на вашем компьютере**. MeetFlow Converter ничего не загружает в интернет и не собирает данные — конвертация целиком локальная.

## 💻 Системные требования

- **Windows** 10/11 (64-bit), **macOS** 11+, или современный **Linux** (64-bit).
- ~300 МБ на диске. Интернет для работы **не нужен**.
